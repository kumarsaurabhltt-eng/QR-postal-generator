# QR Postal Receipt Generator

## STEP 1: Install Requirements
Open VS Code Terminal and run:
pip install -r requirements.txt

## STEP 2: Prepare CSV
Create "input.csv" with this content:
tracking_number
ABC1001
ABC1002
ABC1003

## STEP 3: Run the Project
python app.py input.csv output/report.pdf

## STEP 4: Output
Check the "output" folder for the generated PDF receipts.
